    公開年毎にアニメテレビレギュラーシリーズ数を集計する #

     1PREFIX schema: <https://schema.org/>
 2PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
 3PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
 4PREFIX class: <https://mediaarts-db.bunka.go.jp/data/class#>
 5SELECT ?y (COUNT(DISTINCT *) AS ?cnt)  WHERE {
 6  ?s a class:AnimationTVRegularSeries ;
 7     schema:datePublished ?datePublished .
 8}
 9GROUP BY (SUBSTR(?datePublished, 1, 4) AS ?y)
10ORDER BY ASC(?y)
